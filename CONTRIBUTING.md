# Contributing to Multiple-Myeloma-Risk-Classifier-MMRC-

Thank you for considering contributing!  
This project is a machine learning classifier for risk stratification in multiple myeloma using RNA-seq gene expression and clinical data. Any improvements, bug fixes, feature suggestions, or documentation updates are very welcome.

## Ways to Contribute
- Report bugs or unexpected results (especially with different datasets)
- Suggest improvements (e.g., better models, feature selection, hyperparameter tuning, visualization)
- Add support for new input formats or additional performance metrics
- Improve documentation or add example datasets (anonymized if possible)
- Fix typos, code style, or small optimizations

## How to Submit Changes
1. **Fork** this repository.
2. Create a new branch: `git checkout -b feature/improve-model` or `git checkout -b fix/bug-description`
3. Make your changes.
4. Commit with clear messages: `git commit -m "Add ROC curve plotting"`
5. Push to your fork: `git push origin feature/improve-model`
6. Open a **Pull Request** against the `main` branch.
   - Describe what you changed and why.
   - If it's a big change, open an issue first to discuss.
